# Repo del módulo de backend con nestjs

## Comandos clase 1

Requisitos:

- npm

- node

```bash
# instalar nestjs
npm install -g @nestjs/cli

# revisar la versión
nest --version
```

```bash
# Crear un proyecto
nest new productos-backend

# entrar al proyecto
cd productos-backend

# abrir el vscode
code .

# correr el proyecto
npm run start
```
